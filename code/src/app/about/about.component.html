About CryptoShakti

Welcome to CryptoShakti—a decentralized lending and investing platform that empowers individuals to access funds at minimal interest rates while providing investors an opportunity to earn reliable returns.

Designed with transparency and efficiency in mind, CryptoShakti integrates advanced blockchain technologies to revolutionize decentralized finance.

Key Features

  • Low-interest borrowing options tailored to individual needs.
  • Investment schemes offering competitive returns—Bronze, Gold, and Diamond tiers.
  • Secure payment systems using MetaMask wallet.
  • Fraud prevention through automated smart contracts.
  • Seamless user authentication with Analog Watch SDK.

Technologies Used

  • Angular: Frontend framework for building the user interface.
  • Firebase: Backend service for storing user data securely.
  • MetaMask: Cryptocurrency wallet for handling payments.
  • Smart Contracts: Decentralized logic for secure and automated transactions.
  • Analog Watch SDK: Decentralized identity verification and authentication.
